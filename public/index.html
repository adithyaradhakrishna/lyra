🚀 How to Install & Setup

1

Download or Install

Either download the .vsix file above OR install directly from VS Code Marketplace

2

Install from VSIX (if downloaded)

In VS Code, go to Extensions → Click "..." menu → "Install from VSIX" → Select lyra-0.0.1.vsix

3

Set Your API Key

Create a .env file in your project root and add your Groq API key:

GROQ_API_KEY=your_api_key_here
4

Start Coding

Open a Python file and LYRA will automatically activate. Look for the LYRA icon in your VS Code activity bar!

5

Get Help

Click the LYRA Mentor sidebar to see AI-powered suggestions and encouragement as you code

⚙️ Configuration

LYRA uses Groq API for intelligent code analysis. Get your free API key:

Get Groq API Key
